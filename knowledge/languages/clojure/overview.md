# Clojure Overview

Clojure is a dynamic Lisp on the JVM focused on immutable data and interactive development. TStack records this as practical engineering guidance for maintainable production work.

## Core engineering model

- Match Clojure to the runtime, team skill, ecosystem, and operational target.
- Keep domain logic separate from platform-specific integration code.
- Prefer idiomatic package, build, and test tooling.
- Document supported versions, runtime assumptions, and deployment constraints.

## Appropriate use

Use Clojure when its ecosystem, runtime model, or deployment environment directly supports the project goal. Avoid choosing it only for novelty.
