# Rust Language Pack

This pack provides production-oriented Rust guidance for TStack's Engineering Knowledge Base.

It focuses on safety, performance, secure systems design, testing, and release readiness.

## Topics

- [Overview](overview.md)
- [Security](security.md)
- [Performance](performance.md)
- [Testing](testing.md)
- [Production](production.md)

## Pack Boundary

This pack covers general Rust. Guidance for Tokio, Actix, Axum, embedded Rust, WebAssembly, and cryptographic libraries should live in dedicated framework or domain packs.
