# Python Language Pack

This pack defines the first maintainable knowledge unit for TStack's Engineering Knowledge Base.

It focuses on production Python engineering rather than syntax tutorials. The goal is to give TStack enough structured guidance to support future explainable recommendations, project scoring, and language comparison.

## Topics

- [Overview](overview.md)
- [Security](security.md)
- [Performance](performance.md)
- [Testing](testing.md)
- [Production](production.md)

## Intended Use

TStack should use this pack to reason about:

- Python project structure.
- Runtime and packaging expectations.
- Common security risks.
- Testing quality gates.
- Performance tradeoffs.
- Production readiness.

## Pack Boundary

This pack covers general Python engineering. Framework-specific guidance for Django, FastAPI, Flask, PyTorch, Airflow, and similar ecosystems should be added as separate framework packs.
