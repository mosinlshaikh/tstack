# Swift Overview

Swift is a modern language used for iOS, macOS, Apple platform apps, and some server-side services.

## Strengths

- Strong safety-oriented language design.
- First-class Apple platform support.
- Good performance characteristics.
- Modern concurrency features.

## Tradeoffs

- Apple platform release workflows add operational complexity.
- Build and signing configuration can be fragile.
- Ecosystem varies outside Apple platforms.

TStack should prefer `REVIEW` when platform target, tests, signing, or dependency resolution are unclear.
