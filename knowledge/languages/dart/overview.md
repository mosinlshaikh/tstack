# Dart Overview

Dart is a client-optimized language commonly used with Flutter for mobile, desktop, and web applications.

## Strengths

- Strong Flutter integration.
- Productive UI development.
- Sound null safety.
- Good tooling for mobile and cross-platform apps.

## Tradeoffs

- Ecosystem is strongest around Flutter.
- Platform-specific behavior still requires native review.
- Release workflows depend on target stores and platforms.

TStack should prefer `REVIEW` when SDK constraints, tests, lockfile, or release target are unclear.
