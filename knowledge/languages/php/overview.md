# PHP Overview

PHP is a server-side language widely used for web applications, content systems, APIs, and business applications.

## Strengths

- Mature web ecosystem.
- Simple deployment on many hosts.
- Strong frameworks and Composer dependency management.
- Large pool of operational knowledge.

## Tradeoffs

- Legacy codebases may lack modern structure.
- Security depends heavily on framework and input-handling discipline.
- Runtime and extension differences matter.

TStack should prefer `REVIEW` when PHP version, Composer lockfile, tests, or framework configuration are unclear.
