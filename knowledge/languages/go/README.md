# Go Language Pack

This pack provides production-oriented Go guidance for TStack's Engineering Knowledge Base.

It focuses on service engineering, concurrency discipline, testing, security, performance, and deployment readiness.

## Topics

- [Overview](overview.md)
- [Security](security.md)
- [Performance](performance.md)
- [Testing](testing.md)
- [Production](production.md)

## Pack Boundary

This pack covers general Go. Guidance for Gin, Echo, gRPC, Kubernetes controllers, Terraform providers, and other ecosystems should live in separate framework packs.
