# C++ Overview

C++ is a systems language used for performance-sensitive applications, infrastructure, games, real-time systems, and native libraries.

## Strengths

- High performance and low-level control.
- Strong abstraction capabilities.
- Large existing ecosystem.
- Suitable for systems with strict latency or resource constraints.

## Tradeoffs

- Complexity is high.
- Undefined behavior and lifetime errors remain serious risks.
- Build systems and dependency management can be difficult.
- ABI and compiler differences matter.

TStack should prefer `REVIEW` when C++ standard, build strategy, tests, or static analysis are unclear.
