# Perl Overview

Perl is a text-processing and systems scripting language used in automation and legacy web systems. TStack treats this pack as a practical engineering guide, not a full language specification.

## Core engineering model

- Understand the runtime and deployment target before choosing Perl.
- Prefer idiomatic project structure and standard package tooling.
- Keep business logic testable and separate from infrastructure boundaries.
- Document version requirements, supported platforms, and operational assumptions.

## Common use cases

- Application development where the ecosystem is a good fit.
- Automation, services, or domain-specific tooling when maintainability is clear.
- Integration with existing teams, libraries, and runtime infrastructure.
